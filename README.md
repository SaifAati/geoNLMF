## Installation
To install geoNLMF from PyPI:

    pip install geoNLMF

Building from source:

    git clone...
    cd geoNLMF
    pip install -e .


### Dependencies

#### Core requirements
*Python 3* and C++>11. The package is tested on 3.6+ and C++11. 

[numpy](http://www.scipy.org) , [scipy](http://numpy.org), [matplotlib](http://matplotlib.org).

#### Recommended
[GDAL](http://gdal.org).
